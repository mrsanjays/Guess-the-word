# Guess-the-word

Problem Statement
In this project, you have to write a computer program to create an interactive game application called Guess the Word. The game must have the following major components:

A repository of English words.

A function to jumble the letters of the English words.

Display a word such that its letters are jumbled, its part of speech and its meaning. Then prompt the player to guess the word one-by-one.

Check whether the player had guessed the correct word.

Keep a count of the player's correct guesses.
